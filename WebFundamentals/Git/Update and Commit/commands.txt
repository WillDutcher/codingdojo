1. Go to Github URL and fork project
2. Go to my remote copy and get URL to clone
3. Make a directory to maintain project
4. git clone <url>.git
5. cd git_demo
6. vi demo_index.html
7. Add my info; save and quit
8. cd ..
9. git add .
10. git commit -m "Add my info"
11. git push (to my remote copy)
12. Go back to original URL of project I forked and create pull request with commit message
